Class: APNGmaker

(async) APNGmaker(images, FPSopt, numplaysopt) → {blob}

generate asynchronously a APNG blob from an array of canvas

Constructor

(async) new APNGmaker(images, FPSopt, numplaysopt) → {blob}

Parameters:
Name Type Attributes Default Description
images Array.<HTMLCanvasElement> array of canvas as source for the Animated PNG
FPS number <optional>
 30 the expected FPS
numplays number <optional>
 0 set numbers of plays, 0 for infinite loop

Example
//create a APNG blob from images (array of canvas) FPS of 60 and with 1 play
var APNGblob = await new APNGmaker(images,60,1)
var url = URL.createObjectURL(APNGblob)
//create link for downloading as myAPNGfile.png
var a = document.querySelector("a")
a.setAttribute("download","myAPNGfile.png")
a.href = url
//display the APNG in browser
var img = document.querySelector("img")
img.src = url
await img.decode()
